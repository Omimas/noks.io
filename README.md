# noks.io
Noks.io, üniversite ve lise öğrencileri için anonim ve güvenli bir topluluk platformudur. Kendi kampüsünüzde veya sınıfınızda fikirlerinizi paylaşabilir, sohbet edebilir ve topluluğun nabzını tutabilirsiniz. AI destekli moderasyon sayesinde içerikler güvenli, moderatörler topluluğun kontrolünde.
